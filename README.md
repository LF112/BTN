## 介绍

- `BTNEXT` 是基于 `react` 开发的 `BT-Panel Plugin` 旨在提供更好的面板体验。

  - 完整的面板界面、操作
  - 扁平风格
  - 按需加载
  - ......

- 技术栈
  - 前端框架 [React](https://reactjs.org/)
  - 构建工具 [Vite](https://vitejs.dev/)

> `BTN` ≈ `BTCO 3.0`，`BTN` 实则是在继 `BTCO` 发布后而重新构建的，更加全面的响应式面板插件的下一个版本。
